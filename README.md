## Workflow
1. workspace --> to create, debug, and manage software.

2. virtual environment
    - python -m venv .venv
    - add .gitignore --> *
    - activate
    <br>

3. pip install -r requirements.txt
    - python.exe -m pip install --upgrade pip
    <br>

4. setup project:
    - cd src
    - django-admin startproject mkmHome
    - cd mkmHome
    - python manage.py runserver
